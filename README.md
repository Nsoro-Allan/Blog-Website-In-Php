# Installing
-Open Xampp
-Open PhphMyAdmin
-create a database
-Import the database in the Database Folder

# Running
-Place the folder in the Htdocs folder in the xammp folder
-open your browser
-write "localhost/" then after the slash enter the name of your project folder.
